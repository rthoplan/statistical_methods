Installation

R is the core language. Select your operating system to see the instructions.

  1. Go to the CRAN website and select a mirror location.
  2. On the mirror page, click "Download R for Windows".
  3. Click on the "base" link for first-time installs.
  4. Click the main "Download R-x.x.x" link at the top.
  5. Run the downloaded `.exe` file and follow the wizard.
  1. Go to the Comprehensive R Archive Network (CRAN).
  2. Click on the "Download R for macOS" link.
  3. Find the latest release package (`.pkg` file) for your macOS version.
  4. Download and run the `.pkg` file. Follow the on-screen instructions.

Your First Steps in R

Open the R console after installation. It's where you type commands. Let's try some.

1. R as a Calculator

> 2 + 2
[1] 4

2. Creating Variables

> my_variable <- 10
> my_variable * 5
[1] 50

You're All Set!
